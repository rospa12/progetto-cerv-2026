# MASTER PLAN CERV-2026 - Pay Transparency

Dashboard istituzionale realizzata con Streamlit per il progetto CERV-2026, dedicata al tema della trasparenza retributiva.

## Contenuti della dashboard

- Titolo istituzionale del progetto
- Countdown alla data del 7 giugno 2026
- KPI target:
  - 150 PMI
  - 500 HR Manager
  - 5.000 Lavoratori
- Tabella con 5 Work Package e scadenze (18 mesi)
- Sezione "Note per i Partner":
  - Veneto Lavoro
  - AIDP
  - Matrix
- Sidebar con riferimenti normativi UE

## Tecnologie

- Python
- Streamlit
- Pandas

## Avvio in locale

1. Installa le dipendenze:

```bash
pip install streamlit pandas
```

2. Avvia l'app:

```bash
streamlit run app.py
```

## Deploy su Streamlit Community Cloud

1. Carica il progetto su GitHub.
2. Vai su [Streamlit Community Cloud](https://streamlit.io/cloud).
3. Collega il repository `rospa12/progetto-cerv-2026`.
4. Imposta come file principale `app.py`.
5. Avvia il deploy e condividi il link pubblico.
